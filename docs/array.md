## 배열
배열은 알고리즘에서 많은 양의 데이터를 저장하고 유지하기 위하여 사용하는 것이다. 정적인 자료 구조로 기억 장소의 추가가 어렵고, 데이터 삭제 시 데이터가 저장 되어 있던 기억장소는 빈 공간으로 남아있어 메모리의 낭비가 발생한다.
예로는 1년간의 일일 입장객 수, 상품의 월별 매출액, 모든 회원 가입자의 이름 등 많은 데이터를 다룬다.
배열의 원칙으로는 담을 각각의 데이터들은 반드시 같은 종류여야만 한다. 
배열은 데이터 타입이 동일한 변수를 일직선 상에 늘어 놓은 것으로, 배열 자체에 데이터 타입이 있다.


변수와 마찬가지로 다른 변수 및 배열과 구분되고 원하는 배열을 선택할 수 있도록 배열명을 붙인다.
**배열명 붙일 때 주의해야할 점**
1. 배열명은 고유해야 한다. (배열명 중복x )
2. 숫자로 시작하거나 숫자 이름으로 시작하는 배열명은 사용할 수 없다.
   

- 배열 요소들의 갯수를 배열 요소 수라고 한다. 지정된 배열 요소를 가리킬 때 요소 번호를 배열의 첨자라고 부른다.
- 첨자로 데이터에 접근이 가능하다.
- 반복적인 데이터 처리 작업에 적합한 구조이다.
- 배열은 데이터마다 동일한 이름의 변수를 사용하여 처리가 간편하다.
- 배열은 사용한 첨자의 개수에 따라 n차원 배열이라고 부른다.

실제 데이터에서는 전체 크기를 가늠하기 힘든 때가 많아 너무 적은 영역이 할당 되거나 너무 많은 영역이 할당되는 문제가 있어 효율적이지 않다. 이런 고민을 해결하기 위해 크기를 지정하지 않고 자동으로 리사이징하는 배열인 동적 배열이 등장했다.

## 문자열

문자열이란 문자 코드 시스템의 '연속된 문자들'로  표현한다. 각 요소에 문자가 포함된 문자 타입 배열이다. 
배열에 저장된 문자 데이터 만으로는 그 배열의 몇번째 요소에서 문자열이 끝나는지 알 수 있는 방법이 없다.
예를 들어 10개의 요소가 있는 배열 str로 "HELLO"라는 문자열을 표한한다고 할 때, str[5]이후로는 사용하지 않는 방법을 표시하기 위해 
문자열 길이 변수를 준비하거나 문자열 끝에 보초값을 저장하는 방법을 들 수 있다.

#### 문자열 길이 변수를 준비하는 방법
배열 속 문자열의 길이를 정수 타입 변수에 저장하는 방법. "HELLO"의 경우 문자길이 변수에 5라는 값이 저장이 되고 이 변수의 값을 참조하면 배열 str에 저장된 문자열의 길이는 5자이며, 유효한 배열 요소는 str[0]~str[4]라는 것을 알 수 있다.

#### 문자열 끝에 보초 값을 저장하는 방법
문자열 구성 문자로 절대 표시되지 않는 문자코드를 문자열 끝 요소 뒷 부분에 저장하는 방법. 문자열 길이를 저장하는 변수가 필요 없다.
보초 값으로는 0(문자 '0'이 아닌 숫자 0)을 사용하는 것이 일반적이다.
"HELLO"는 'O'의 다음 요소 위치 str[5]에 값 0을 저장한다. 그러면 값 0ㅇ을 발견한 시점에서 문자열의 구성 문자의 끝을 차례대로 확인할 수 있다.

## 관용적으로 사용되는 변수명
1. 반복문의 반복횟수를 저장하는 변수 : 반복 처리하는 알고리즘에서 반복 횟수를 유지하는 정수 타입 변수명 값으로 i,j,k가 자주 사용된다. 과거에 Fortan이라는 프로그래밍에서 데이터 타입을 파악할 수 없는 변수들은 암묵적으로 정수 타입 변수로 생각했었던 흔적으로 추정된다.
2. 배열 첨자로 사용되는 변수 : index,idx
3. 수를 세는데 사용되는 변수 : counter, count, cnt
4. 문자열을 다루는 변수 (배열): str,string

### JS
다른 언어의 배열과 다르게 어떤 타입의 데이터라도 넣을 수 있다.
배열의 첫 첨자에 문자열, 두번째에는 숫자, 세번째에는 객체를 넣는 식으로 사용할 수 있다.
ECMA Script가 동적으로 크기를 조절하므로 데이터를 추가하면 자동으로 커진다.

배열 만드는 방법
1. Array 생성자를 쓰기
``` javascript
var colors = new Array();
```

2. 배열에 데이터가 몇 개 들어갈지 알고 있다면 생성자에 매개변수를 넘겨서 length 프로퍼티가 자동으로 그 값에 맞게 바뀌게끔 할 수 있다.

``` javascript
var colors = new Array(20);
```

3. 배열에 들어갈 데이터를 넘겨도 된다.
``` javascript
var colors = new Array("red","blue","green");
```
